📖 How to Use Integration Tools:

  1. Generate AI translations using the Suggestions tab
  2. Upload your current translation file (en.json or es.json)
  3. Click "Merge with AI Translations" to combine them
  4. Download the merged file and replace your original
  5. Test your app to ensure everything works correctly

🤖 AI Translation

Generate full translations using OpenAI API. Enter your API key and let AI translate all missing keys.

📖 How to use AI Translation:

  1. Get an OpenAI API key: Visit platform.openai.com/api-keys to create one
  2. Enter your API key: Paste it in the field below (starts with "sk-")
  3. Choose settings: Select your preferred model and translation style
  4. Estimate cost: Click "Estimate Cost" to see the expected API usage cost
  5. Translate: Click "Translate All Missing Keys" to generate AI translations
  6. Review & Download: Review the translations and download them as JSON files
