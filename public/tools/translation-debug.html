üöÄ Quick Actions

Copy these commands to your browser console when on your Angular app:

1. Collect Missing Translations

// Run this in your Angular app's browser console const missing = JSON.parse(localStorage.getItem('missingTranslations') || '[]'); console.log('Missing translations:', missing);

2. Clear Missing Translations Log

// Clear the missing translations log localStorage.removeItem('missingTranslations'); console.log('Missing translations log cleared');

3. Get Statistics

// Get statistics about missing translations const missing = JSON.parse(localStorage.getItem('missingTranslations') || '[]'); const byLanguage = missing.reduce((acc, item) => { if (!acc[item.language]) acc[item.language] = []; acc[item.language].push(item.key); return acc; }, {}); const uniqueKeys = [...new Set(missing.map(item => item.key))]; console.log('Statistics:', { total: missing.length, unique: uniqueKeys.length, byLanguage });
